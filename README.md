# VentoyISOManager

A Tauri-based application for managing Ventoy ISO images, providing tools to download new ISOs, remove old ones, identify distribution types, calculate hashes, and analyze ISO contents.

## Features
- Easy ISO image management
- Simple boot configuration
- Support for multiple operating systems
- User-friendly interface

## Getting Started
1. Clone the repository
2. Install dependencies with pnpm
3. Run the application using `pnpm tauri build`

## Usage
The tool allows you to:
- Add, remove, and organize ISO images
- Configure boot settings
- View system information
